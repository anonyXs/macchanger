# Tools and Scripts

**macchanger** allows you changing your current MAC of any interface.
</br>
Works with many Linux distributions.
</br >
Tested on Debian / Kali
</br >
</br >
**$ Make it executable**
</br >
> chmod +x macchanger.sh
</br >
</br >
**$ Usage**
</br >
> ./macchanger \<INTERFACE> \<NEW-MAC XX:XX:XX:XX:XX:XX>
